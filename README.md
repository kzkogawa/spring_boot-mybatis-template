# Template project using Spring boot & mybatis

## Some portions implemented in this project are in below.
#### view
* thymeleaf
#### controller
* mvc
* api
#### database
* mysql
#### orm
* mybatis
#### security
* authentication
* csrf
#### log
* logback

* mybatis-generatorConfig.xml 
* src/main/java/demo/config/data/DataConfig.java 
And re-run mybatis generator!
